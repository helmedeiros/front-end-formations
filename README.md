# front-end-formations

Code written while working through Code School's course on modern **HTML5 & CSS3**.

A learning record covering the HTML5 doctype, new semantic tags, form elements
and input types, and CSS3 features such as border-radius, box-shadow and
gradients. Kept for the historical trail.

## Layout

Each `level_N/` folder holds the exercises for that level, one numbered
sub-folder per exercise.
